Hi, I'm Bryn.

I wrote the search engine for the Open Directory Project. That was back when Gigabytes were considered a large unit of measure. After that I improved on the same software to handle 20 times as much data to power the news search for Topix.

After Topix I co-founded Blekko and helped build a web-scale search engine. Working on everything from crawling to classification.

I created Bottlecount an online wine cellar tracking system.
